# Piscologo

    https://github.com/josue-balbontin/Piscologo

## Estructura

- `Code` : Contiene el código fuente del proyecto.
- `docs` : Contiene la documentación del proyecto.
- `video`: Contiene video del proyecto

## Ejecutar codigo 

Para ejecutar el proyecto, sigue estos pasos:
1. Asegúrate de tener instalado Node.js y Angular CLI en tu máquina.
2. Navega al directorio `Code` en tu terminal.
3. Ejecutar el comando `npm install` para instalar las dependencias del proyecto.
4. Una vez instaladas las dependencias, ejecuta el comando `npm start` para iniciar el servidor de desarrollo.
